# querymorewords
一个简单的爬虫程序，用于查询单词。</br>
写这个程序的动机是要背考纲的单词，高考考纲上是没有单词释义的，就想写一个程序一次性把单词释义查出来。</br>
从同目录下的sample.txt文件按行读取单词并从有道词典爬取单词释义输出。</br>
可用PowerShell等直接读到查询结果。</br>
sample.txt中已经放了A开头的考纲单词用于测试。</br>
